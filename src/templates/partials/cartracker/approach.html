The project is divided into two applications. The Android Application which is responsible for collecting the information from the car and uploading it. The Web Application which collects the information from the Android app performs analysis and presents it to the user in an easy to read and understand format.

To keep things simple and allow easy code re-use between the Android application and Web app both were written in Java.

The webapplication utilizes Spring Framework for hanlding dependency injection, the RESTFul API, serving WebPages using Thymeleaf as a templating engine, and to secure the application. MyBatis was used to provide a simple Data Access Layer between the application and its database.

On the android side piers odb-java-api is used to communicate with the vehicle. Spring RestTemplate is used to consume the RESTFul api exposed by the web application.
