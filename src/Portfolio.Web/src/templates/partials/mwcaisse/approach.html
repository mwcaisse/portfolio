I wanted a site that was responsive and quick to load, which led me to choose using Bootstrap and Nunjucks. Bootstrap for making a responsive site that looked nice and Nunjucks for the templating benefits it provided.

The site consists of two template pages both backed by data stored in JSON format. This allows for adding similarly formatted content without duplicating coding.

The contact portion of the site is powered by a simple .NET Core Rest API. I used .NET Core for two reasons: I wanted to learn more about .NET Core and for the ease of setup and configuration.
