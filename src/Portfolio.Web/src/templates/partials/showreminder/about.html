Show Remidner is a web application to tell you the next air date of TV shows that you are following.

The main reason for starting the project was to learn .NET Core, Angular 2, and Typescript and I always felt that I was forgetting to keep up to date on a show. I thought why not learn something new and make my life a little bit easier.

The project uses the The Movie DB to pull show information and thier next air date.
