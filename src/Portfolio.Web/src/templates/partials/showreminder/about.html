Show Reminder is a web application to remind you when a TV Show is airing next or when it has last aired. The user can add shows to track and when visting the website the last + next air dates are shown. I always felt like I was forgetting to watch a certian episode of a show, this is the result. In the future it might have support for automatically sourcing a place to watch the episode.

I wanted to learn Angular 2, Typescript, and .NET Core, this provided a great oppurtunity to learn both.
